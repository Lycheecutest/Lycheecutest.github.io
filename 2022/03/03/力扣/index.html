判断字符串是否唯一

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。

限制：

  • 0 <= len <= 100
  • 可以考虑不使用额外的数据结构
  1. 使用双重循环
public boolean isUnique(String astr) {
    if(astr.length() > 26) return false;
    
    for(int i = 0; i < astr.length(); i++){
        for(int j = i + 1; j < astr.length(); j++){
            if(asrt.charAt(i) == astr.charAt(j)) return false;
        }
    }
    return true;
}
  1. 用数组记录出现过的字符
    public boolean isUnique(String astr) {
        astr.toLowerCase();
        int[] str = new int[28];

        for (int i = 0; i< astr.length(); i++){
            int var = astr.charAt(i) -'a';
            if(str[var] == 0) str[var] = 1;
            else return false;
        }
        return true;
    }
/** 使用了toLowerCase()后，字符全部变成小写，使用后无法获得原字符串**/
  1. 使用位运算，拿一个整数记录出现过的字符
public boolean isUnique(String astr) {
	int hash = 0;
    for(char ch: astr.toCharArray()){
        int index = 'ch' - a;
        if(hash & (1 << index) != 0){
            return false;
        }else hash |= (1 << index);
    }
    return true;
}
/** 问题： 如果字符包括大小写，int是32位的，就会存不下，如果字符的范围扩大到ASCII表，该方法就无法使用了**/

收获

  1. Java中String类型是引用类型，不能像Char数组那样直接用下标去取出对应的字符。
  2. String类型的两个常用方法： char[ ] toCharyArray() ，String对象调用该方法后，返回一个字符数组。 char charAt(int index) ，String对象调用该方法后，返回对应下标的字符。

判断是否互为字符重排

给定两个字符串 s1 和 s2，请编写一个程序，确定其中一个字符串的字符重新排列后，能否变成另一个字符串。

说明：

  • 0 <= len(s1) <= 100
  • 0 <= len(s2) <= 100
  1. 对两个字符串进行排序后一个一个进行比对
    public boolean CheckPermutation(String s1, String s2) {
        if(s1.length() != s2.length()) return false;

        char[] chars1 = s1.toCharArray();
        char[] chars2 = s2.toCharArray();
        Arrays.sort(chars1);
        Arrays.sort(chars2);

        for(int i = 0; i< s1.length(); i++){
            if(chars1[i] != chars2[i]) return false;
        }
        return  true;
    }
}

可以不用排序，直接使用Arrays.equals(s1,s2)判断两个字符数组是否相等

  1. 用Ascii码来比较，当两个字符串长度相等，所有字符的ASCII码相等，那么两个字符串就互为字符重排
 public boolean CheckPermutation(String s1, String s2) {
        int sum1 = 0;
        int sum2 = 0;
        if(s1.length() != s2.length()) return false;
        else{
            for(int i=0; i<s1.length(); i++){
                sum1 += s1.charAt(i);
                sum2 += s2.charAt(i);
            }
        }
        return (sum1 == sum2); 
    }
/**如果出现 ac 和 bb那么两个字符数组的Ascii值是一样，但是不是互为字符重排。所以该方法不对**/

收获

  • Arrays工具类的使用

URL化

URL化。编写一种方法，将字符串中的空格全部替换为%20。假定该字符串尾部有足够的空间存放新增字符，并且知道字符串的“真实”长度。（注：用Java实现的话，请使用字符数组实现，以便直接在数组上操作。）

提示：

  • 字符串长度在[0,500000]范围内。

一、遍历字符串，遇到空格替换成”%20”

    public String replaceSpaces(String S, int length) {
        StringBuffer sb = new StringBuffer();
        for(int i=0; i<length; i++){
            if(S.charAt(i) == ' '){
                sb.append("%20");
            }else{
                sb.append(S.charAt(i));
            }
        }
        return sb.toString();
    }

这里不能用String类型去存储新的字符串，因为String类型是无法改变的，每次改变字符串就是创建一个新的字符串，会花费很多的时间。如果需要对字符串进行多次修改，需要使用StringBuffer&StringBuilder类
